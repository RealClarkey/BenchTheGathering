# Core Game Design Document

## Game Rules

## Starting The Game
- Draw 5 cards (minimum 1 hero or re-draw)
- Choose initial hero card for battle field
- No hero's on field lose game (this could be changed for a commander style game)

## Player has X amount of HP at start of game

## Turn Structure
- Draw 1 card
- Play 1 mana card
- Play 1 hero or skill
- Attack phase
- End Turn

## Battlefield
- Max 3 Heros active per player

## Mana System
- Mana can be used once only
- (or mana cards increase a max mana, that refills each turn?)

## Combat
- Heroes attack heroes or player (if commander style)
- Damage is simultaneous

## Type System
- Fire > Nature
- Nature > Tech
- Tech > fire
- Neutral = no advantage

## Buff System
- Buffs are applied via skill cards
- Buffs display as icons on cards

## Evolution
- Heroes evolve after a trigger condition
- Access to evolution abilities
