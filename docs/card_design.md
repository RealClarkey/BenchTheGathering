# Card Design Document

# Used as a reference for design choices.

## Card Types:
- Hero
- Mana
- Skill

----------

## Hero Card:
- Name
- Hero Type (Fire, Nature, Tech, Neutral)
- Hit Points
- Pic (Image)
- Buffs (Icons)
- Abilities (description, attack damage, mana value)
- Evolution Abilities

----------

## Mana Card:
- Mana value.

----------

## Skill Cards:
- Mana cost.
- Skill Type (Buff or Instant)
- Target Type (Self, Ally or Enemy)
- Duration (Instant, Timed, Permanent (buff) )
- Effect (ability)
